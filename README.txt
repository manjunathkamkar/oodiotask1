# OODIO_Task1

OODIO_Task1 is a Python Script to placing logo and QRcode in a Template file.

## Installation

Step 1 :  #Create Virtual Environment to run all the dependencies.
			pip install virtualenv

Step 2 :  #  Install 'Make file', which will internally do the required installation for all the dependencies.
			make install

Step 3 :  # Run Make file  which will run the 'tempCreator.py'
			make launch
